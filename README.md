# SimAcademy Blog

Blog de SimAcademy — https://blog.simacademy.lat

## Stack

- HTML + Tailwind CSS (CDN, sin build step)
- GitHub Pages para publicación automática
- Dominio: blog.simacademy.lat

## Archivos de marca

Logos y manuales de imagen disponibles en `assets/logos/` y `assets/brand/`.

## Publicación

El blog se publica automáticamente via GitHub Pages desde la rama `master`.

Para habilitar GitHub Pages:
1. Ir a Settings → Pages en el repo de GitHub
2. Source: Deploy from a branch → master
3. Custom domain: blog.simacademy.lat

## Desarrollo local

```bash
# Servir con Python
python -m http.server 8000
```
